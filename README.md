# chill pickle

Personal website and CV generator built with Svelte 5 + Vite.

## Dev

```bash
npm run dev
npm run build
npm run preview
```

## Lint

```bash
npm run lint
```

## Structure

- `src/cvData.js` — CV content
- `src/CV.svelte` — CV renderer
- `src/pages/AboutMe.svelte` — About page with CV download
- `src/pages/News.svelte` — News feed
